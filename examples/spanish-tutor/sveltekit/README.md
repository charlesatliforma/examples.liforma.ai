# Spanish Tutor — SvelteKit

## Run

```bash
npm install
cp .env.example .env
npm run dev
```

http://localhost:3007

## Verify

```bash
npm run verify
```

## Customise

- `src/lib/lessons.ts` — lesson content and experience IDs
- `src/routes/+page.svelte` — main lesson UI
- `src/lib/components/LiformaEmbed.svelte` — SDK + web component

## Environment

See `.env.example`.
